# Componentes — Gym & Fitness Landing

Mapeo de secciones a componentes de la biblioteca.

---

## Hero Section

**Componentes recomendados:**
- `btn-get-started-gradient` — CTA "Start free trial" (cambiar gradiente a verde neón)
- `btn-explore` — Boton "Watch classes" estilo outline neón
- `bg-night-sky` — Fondo con estrellas para hero nocturno
- `bg-gradient-grid` — Gradient grid como fondo energetico
- `anim-glow` — Glow pulsante para badge "Open 24/7"

**Adaptacion:**
- Gradiente del boton a linear-gradient(135deg, #00ff88, #00cc6a)
- Texto del boton en negro (#0a0a0c)
- Hover: box-shadow glow rgba(0,255,136,0.4)
- Heading en Bebas Neue con text-shadow neón

---

## Classes Section

**Componentes recomendados:**
- `card-cyber` — Card 3D interactive con grid trackers, glow — perfecto para clases intensas
- `card-dynamic` — Dynamic Hover Card con 3d, shine, glow para clases
- `card-features` — Features card con layers expandibles para descripcion de clase
- `card-3d` — 3D Hover Card para foto de la clase con tilt

**Adaptacion:**
- Surface #15151a, border 1px #1a1a20, radius 12px
- Hover: border neón rgba(0,255,136,0.3) + glow 0 0 24px rgba(0,255,136,0.15)
- Badge intensidad: border neón, uppercase, weight 700
- Nombre clase en Bebas Neue uppercase

## Trainers Section

**Componentes recomendados:**
- `card-profile` — Card de perfil para trainer con avatar + especialidad + redes
- `card-hover-fill` — Efecto fill en hover para overlay con info del trainer
- `btn-social-hover` — Botones de redes sociales del trainer (Instagram, etc.)
- `card-3d` — 3D card para foto del trainer con tilt en hover

**Adaptacion:**
- Card-profile con foto del trainer, nombre en Bebas Neue
- Hover: card levita + overlay con bio + especialidad
- btn-social-hover circulares con iconos, hover neón
- Especialidad en badge neón

## Pricing Section

**Componentes recomendados:**
- `card-pro-plan` — Pro Plan Card para plan destacado (Monthly)
- `cb-gradient-check` — Checkbox con gradiente (cambiar a neón) para lista de features
- `cb-animated-check` — Check animado alternativo para features
- `tgl-pill` — Toggle pill para switch mensual/anual
- `btn-get-started-gradient` — CTA en cada plan

**Adaptacion:**
- Plan destacado: border neón + glow constante
- cb-gradient-check: gradiente neón, size 16px
- tgl-pill para toggle mensual/anual con track neón cuando checked
- Boton CTA: neón en plan destacado, outline en otros

## Schedule Section

**Componentes recomendados:**
- `card-meeting` — Meeting Card para cada clase del horario
- `card-features` — Features card para dias de la semana expandibles
- `pat-grid-fade` — Grid pattern como fondo de la tabla de horarios

**Adaptacion:**
- Card-meeting adaptada para mostrar clase + trainer + intensidad
- Color-coded por tipo de clase (verde=strength, rojo=HIIT, azul=yoga)
- pat-grid-fade como fondo sutil de la tabla, opacidad 5%
