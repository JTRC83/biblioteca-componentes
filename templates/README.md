# Plantillas web

Catálogo de plantillas de referencia para generación de webs con IA.

## Estructura

Cada plantilla contiene:

- `screenshot.png` — Captura visual de referencia
- `prompt.md` — Descripción en lenguaje natural del diseño
- `design.md` — Design tokens (colores, tipografía, grid, espaciado)
- `components.md` — Mapeo: sección → componente de la biblioteca
- `meta.json` — Metadatos estructurados

## Categorías

- `landing/` — Landing pages completas
- `hero/` — Secciones hero aisladas
- `design/` — Design systems de referencia
