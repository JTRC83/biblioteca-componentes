# Sunset Orange

## Overview
Sistema de design ligero y energetico con tonos naranja vibrantes. Naranja, ambar y blanco cálido. La estetica es dinamica, amigable y llena de energia. Tipografia sans-serif redondeada (Poppins) para titulos e Inter para body. Las superficies son blancos y cremas ambar con bordes sutiles. La sensacion es de una marca moderna y vibrante — una app de delivery, una startup, una marca lifestyle. Bordes redondeados, sombras suaves con tinte naranja, y acentos vibrantes. Sin tonos frios, sin grises apagados. Solo energia y calidez.

## Typography
- **Display Font**: Poppins (sans-serif redondeada, amigable y energetica)
- **Body Font**: Inter (sans-serif)
- **Code Font**: JetBrains Mono

Poppins para headings con peso 700 y 800. Inter para body y UI. El contraste crea sensacion moderna y dinamica. Poppins es redondeada, geometrica y transmite energia.

Type scale: Display clamp(2.5rem, 6vw, 4.5rem) weight 800, Headline 2.25rem 700, Subhead 1.5rem 600, Body 16px 400 line-height 1.6, Small 14px, Caption 12px, Overline 11px 700 uppercase.

## Elevation
Sombras suaves con tinte naranja.
- **Card**: border 1px #fed7aa. Hover: 0 8px 24px rgba(249,115,22,0.12) + lift -3px
- **Focus ring**: 0 0 0 3px rgba(249,115,22,0.20)
- **Modal**: 0 24px 64px rgba(249,115,22,0.18)
- **Nav**: border-bottom 1px #fed7aa

## Components
- **Buttons**: Primary naranja #f97316 texto blanco, radius 12px, padding 12px 24px, weight 600. Secondary: transparent border #fb923c. Ghost: color change. Hover: primary a #ea580c + lift -2px. Gradientes naranja-a-ambar para CTAs destacados.
- **Cards**: Surface blanca, border 1px #fed7aa, radius 16px, padding 28px. Hover: sombra naranja + lift. Estilo dinamico y vibrante.
- **Inputs**: Border 1px #fed7aa, radius 12px. Focus: border #f97316 + ring rgba(249,115,22,0.20).
- **Chips**: Rounded-full, surface-hover bg, naranja active.
- **Navigation**: bg ambar claro, border-bottom naranja claro, 60px. Links Poppins en hover.

## Spacing
- Base: 4px. Scale: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80px
- Section: 80px desktop. Container: 1200px. Gap: 20px

## Border Radius
- 8px: Tags, chips, checkboxes
- 12px: Buttons, inputs
- 16px: Cards, panels
- 24px: Modales, hero cards
- 9999px: Avatares, status dots, chips

## Do's and Don'ts
- Do usar naranja #f97316 como color principal
- Do usar Poppins para headings — transmite energia y amigabilidad
- Do mantener el fondo ambar claro #fffbeb
- Do usar gradientes naranja-a-ambar para CTAs destacados
- Do usar bordes redondeados (minimo 8px)
- Do usar sombras con tinte naranja rgba(249,115,22,0.12)
- Don't usar azules o tonos frios
- Don't usar grises apagados o neutros sin calidez
- Don't usar tipografia serif para headings
- Don't usar border-radius inferior a 8px
- Don't usar neón extremo o glow excesivo
