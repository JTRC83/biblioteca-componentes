# Componentes — Sunset Orange

Mapeo de elementos del design system a componentes de la biblioteca.

---

## Buttons

**Componentes recomendados:**
- `btn-get-started-gradient` — Boton "Get started" con gradiente — perfecto para CTAs energicos
- `btn-buy-now` — Boton "Buy now" (cambiar a naranja vibrante)
- `btn-copy-link` — Boton funcional (adaptar a tonos naranja)
- `btn-send-message` — Boton "Send" para formularios de contacto

**Adaptacion:**
- Primary: gradiente 135deg #f97316 a #fb923c, texto blanco, radius 12px
- Hover: gradiente a #ea580c/#f97316, lift -2px, sombra rgba(249,115,22,0.30)
- Secondary: transparent con border #fb923c
- Ghost: color change a primary en hover
- Cambiar todos los gradientes a tonos naranja-ambar

---

## Cards

**Componentes recomendados:**
- `card-3d` — Card 3D con efecto hover — perfecto para showcases dinamicos
- `card-glass` — Glass card (adaptar a surface blanca con tinte naranja)
- `card-features` — Features card para listados de caracteristicas
- `card-hover-fill` — Card con fill animado en hover — energico y vibrante
- `card-black-friday` — Promotional card para ofertas y descuentos

**Adaptacion:**
- Surface blanca, border 1px #fed7aa, radius 16px
- Hover: sombra naranja 0 8px 24px rgba(249,115,22,0.12) + lift -3px
- Padding 28px para sensacion espaciosa
- Iconos con fondo gradiente #fff7ed a #fed7aa
- Cambiar gradientes de hover-fill a naranja-ambar

## Checkboxes & Radios

**Componentes recomendados:**
- `cb-animated-check` — Check animado (cambiar a naranja vibrante)
- `cb-thumb-like` — Thumb like para feedback positivo — perfecto para lifestyle/social

**Adaptacion:**
- Checkbox: border 1px #fb923c, radius 8px
- Checked: bg naranja #f97316, checkmark blanco
- Focus ring: 0 0 0 3px rgba(249,115,22,0.20)
- Thumb like: color naranja al activar

## Inputs & Forms

**Componentes recomendados:**
- `form-modern` — Formulario moderno (adaptar a tonos naranja)
- `input-floating` — Floating label (naranja en focus)

**Adaptacion:**
- Border 1px #fed7aa, radius 12px, bg blanca
- Focus: border #f97316 + ring rgba(249,115,22,0.20)
- Placeholder: #a8a29e

## Loaders

**Componentes recomendados:**
- `loader-pulse-ring` — Pulse ring loader (cambiar a naranja vibrante)
- `loader-gradient-sphere` — Gradient sphere (cambiar gradiente a naranja-ambar)

## Backgrounds & Patterns

**Componentes recomendados:**
- `bg-sunset-glow` — Sunset glow background — perfecto para hero sections energeticos
- `bg-mesh` — Mesh gradient background (cambiar a tonos naranja-ambar)
- `pat-dots` — Dots sutiles en color naranja claro

**Adaptacion:**
- bg-sunset-glow: gradiente radial #f97316 a #fffbeb, opacidad 15%
- bg-mesh: tonos #f97316, #fb923c, #fbbf24 a baja opacidad
- pat-dots: #fed7aa opacidad 20% como textura sutil
- Mantener fondos ligeros y luminosos
