# Nordic Wood

## Overview
Sistema de design nordico escandinavo. Madera clara, tonos neutros frios, simplicidad y funcionalidad. Estetica hygge: calma, calidez y minimalismo organico. Sage green #5e6b5e + light wood #c4a882 sobre fondo warm off-white #f7f5f2. DM Sans redondeada y humanista. Radios suaves (10-20px). Sombras suaves con tinte calido. La sensacion es de una casa escandinava: luz natural, madera clara, plantas, calma.

## Colors
- **Primary** (#5e6b5e): CTAs, links, elementos interactivos — sage green
- **Primary Hover** (#4d5a4d): Hover sobre primary
- **Secondary** (#c4a882): Acentos, highlights — light wood
- **Background** (#f7f5f2): Fondo warm off-white
- **Background Alt** (#ede9e4): Secciones alternas
- **Surface** (#ffffff): Cards, paneles
- **Border** (#e0ddd7): Bordes sutiles calidos
- **Text Primary** (#2d2d2d): Headings, body
- **Text Secondary** (#7a7a7a): Descripciones, metadata
- **Text Muted** (#a8a8a8): Placeholders, disabled

## Typography
- **Font Family**: DM Sans (Inter fallback)
- **Code Font**: JetBrains Mono

DM Sans redondeada, humanista y calida. Pesos 400, 500, 600. Display weight 500 (no bold excesivo). Body 16px line-height 1.8 (muy generoso). Tracking normal.

Type scale: Display clamp(2rem, 5vw, 3.5rem) weight 500, Heading 1.75rem 600, Subhead 1.375rem 500, Body 16px 400, Small 14px, Caption 12px, Overline 11px 600 uppercase.

## Elevation
Sombras suaves con tinte calido.
- **Card**: border 1px #e0ddd7. Hover: 0 4px 20px rgba(94, 107, 94, 0.06) + lift -1px
- **Focus ring**: 0 0 0 3px rgba(94, 107, 94, 0.1)
- **Modal**: 0 24px 64px rgba(94, 107, 94, 0.08)
- **Nav**: border-bottom 1px #e0ddd7

## Components
- **Buttons**: Primary fill #5e6b5e texto blanco, radius 10px, padding 12px 24px, weight 500, font 14px. Secondary: transparent border 1px #c4a882. Ghost: color change. Hover: primary a #4d5a4d.
- **Cards**: Surface blanca, border 1px #e0ddd7, radius 14px, padding 32px. Hover: sombra suave calida + lift.
- **Inputs**: border 1px #e0ddd7, radius 10px, padding 12px 16px. Focus: border #5e6b5e + ring rgba(94,107,94,0.1).
- **Chips**: border 1px #e0ddd7, radius 9999px, padding 4px 14px, font 12px. Active: fill #5e6b5e texto blanco.

## Spacing
- Base: 4px. Scale: 4, 8, 12, 16, 20, 24, 28, 32, 40, 48, 64, 96px
- Section: 96px (muy generoso). Container: 1200px. Gap: 24px

## Border Radius
- 6px: Tags, chips
- 10px: Buttons, inputs
- 14px: Cards
- 20px: Modales, hero cards
- 9999px: Avatares, pill badges

## Do's and Don'ts
- Do usar sage green #5e6b5e como color principal
- Do usar DM Sans para mantener la calidez
- Do mantener espacios generosos (96px entre secciones)
- Do usar sombras suaves con tinte verde/madera
- Do usar radios suaves (10-20px) para sensacion organica
- Don't usar colores saturados o neón
- Don't usar tipografia angular o condensada
- Don't usar sombras oscuras o pesadas
- Don't usar border-radius menores a 6px
- Don't usar gradients llamativos
